# DocuTray Python Library

[![PyPI version](https://img.shields.io/pypi/v/docutray.svg)](https://pypi.org/project/docutray/)
[![Python versions](https://img.shields.io/pypi/pyversions/docutray.svg)](https://pypi.org/project/docutray/)

The DocuTray Python library provides convenient access to the [DocuTray API](https://docutray.com) from applications written in Python.

## Documentation

See the [DocuTray API docs](https://docs.docutray.com).

## Installation

```sh
pip install docutray
```

### Requirements

- Python 3.10+

## Features

- **Document Conversion**: Convert images and PDFs to text using advanced OCR
- **Document Identification**: Automatically identify document types
- **Data Extraction**: AI-powered extraction of structured data from documents
- **Knowledge Bases**: Organize and search document collections
- **Async Support**: Non-blocking operations for large documents

## Supported Document Types

DocuTray supports extraction from various document types including:

- Invoices and purchase orders
- Financial statements
- Bank and credit card statements
- Payroll and salary slips
- Bills of lading
- Medical prescriptions
- Resumes/CVs
- And more...
